**FiveM-Sniper** is a tool designed to extract various types of server and player data from FiveM servers. 

## Server Data
The tool can extract the following server information:
- **Server Address**
- **Hosting Name**
- **Owner Name**
- **Owner Profile**
- **Owner Avatar**
- **Last Seen**
- **Icon Version**
- **Connect Endpoints**
- **Upvote Power**
- **Burst Power**
- **Support Status**
- **End of Support**
- **Max Clients**
- **Hostname**

## Player Data
It also extracts active player data, including:
- **Name**
- **Discord ID**
- **Steam ID**
